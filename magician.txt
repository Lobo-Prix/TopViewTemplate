# 기본 정보
# 스페이스바를 누르면 다음대사로 넘어간다.
# script 파일은 반드시 UTF-8 인코딩으로 저장되어야 한다.
# 한 줄당 명령어 하나이며, <명령어> <인자> 형태(명령어와 인자 사이에 공백 필수)로 작성한다.
# 프로그램은 빈 줄이 나올때까지 명령어를 계속 실행한다.
# 이미지 파일의 크기는 500*720으로 한다.(해상도가 더 커도 되지만, 종횡비가 맞지 않으면 이미지가 왜곡될 것이다.)
# 
# 명령어 종류
# player_img <이미지파일명>
# -player의 이미지를 인자로 들어온 파일로 교체한다.
# -이미지 파일명은 확장자까지 써야하며, jpg와 png를 지원한다.
# npc_img <이미지파일명>
# -npc의 이미지를 인자로 들어온 파일로 교체한다.
# -이미지 파일명은 확장자까지 써야하며, jpg와 png를 지원한다.
# npc_name <NPC이름>
# -npc의 이름을 인자로 들어온 문자열로 교체한다.
# text <문자열>
# -대화상자의 문자열을 인자로 들어온 문자열로 교체한다.
# next <script>
# -다음장면부터 인자로 들어온 script파일이 진행한다.
# end
# -대화를 종료한다.
# branch <선택지개수> <선택지1> <선택지1의 next script> <선택지2> <선택지2의 next script> ... <선택지n> <선택지n의 next script>
# -선택지 기능. 각 선택별로 다음에 진행할 script파일을 지정해줘야한다.
# -지원 예정
# # <하고싶은말>
# -이 명령어는 무시된다.
player_img girl2.png
npc_img magician.png
npc_name 마술사
whose player
textb Hello World!
textb NPC님 안녕하세요~
whose npc
textb 네. 플레이어님 반갑습니다.
textb 퀘스트를 수락하시겠습니까?
end
